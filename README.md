# GateMatch
GateMatch: Learning Robust Feature Matching with Adaptive Gated Fusion

# GateMatch implementation
Pytorch implementation of GateMatch


## Requirements

Please use Python 3.6, opencv-contrib-python (3.4.0.12) and Pytorch (>= 1.1.0). Other dependencies should be easily installed through pip or conda.

## Explanation

If you need YFCC100M and SUN3D datasets, You can visit the code at https://github.com/zjhthu/OANet.git. We have uploaded the main code on 'core' folder.
